# DPSH PyTorch Implementation

DPSH论文的复现：包含 PyTorch 中 DPSH（深度成对监督哈希）的实现。该项目包括使用两种不同的网络架构（CNN-F 和 ResNet）在 CIFAR 和 NUS-WIDE 数据集上训练和评估深度哈希模型的代码。

- [DPCHash_Baselines](https://github.com/Huenao/DPCHash_Baselines)
